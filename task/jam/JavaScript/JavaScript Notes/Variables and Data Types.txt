Variables

let 
    - block scope
    - cannot be redeclare 
        ex. let x = 0;
        let x = 1; //wrong
        x = 1; //right
var 
    - global scope
    - can be redeclare
        ex. var x = 0;
        var x = 1; //right
        x = 1; //right
const 
    - constant
    - cannot change the initial value
